Vad är programmering? ¶

Datorer har som vi känner dem funnits sedan mitten av 1900-talet. Sen kan man ju fråga sig att vad är en dator? Vi kan definiera en dator som en maskin som kan ta aritmetiska och logiska instruktioner och sedan utföra detta och producera nån form av resultat. En simpel synpunkt är att tänka sig att programering är konsten att skriva instruktioner till datorn som datorn kan uppfatta de. Datorn pratar själv i ett pråk som består bara av 1 och 0. Vi som människor har svårt att kommunisera med detta språk. Därför har man hitta på en massa olika programmerings språk för att underlätta vår kommunikation med datorn. Det finns massa olika programmerings språk och de bra för olika applikationer. En av de mest populära programmering språken nuförtiden är Python och vi kommer att använde Python i detta material.

Vad äro Python? ¶

Utöver att python är en orm så är det också en av de populäraste programmerings språken. Python används till många olika ändamål. Man kan göra websidor, appar, vetenskapliga beräkningar, animationer, spel och egentligen vad som helts annat som man kan tänka sig göra med en dator. Varför är python så populärt? En grund orsakt att python har bygts upp så att de är mycket lästbart och lätt att komma igång med.

Hello, World! ¶

Hejsan välden! Nu ska vi börja med att bekanta oss med Python. Vi kommer att börja med ett simpelt ”Hello, World!” program pågrund av historiska själ. Hello world programmets funktion är att få datorn att ”hälsa” till oss. Programmet är ofta den som man lär sig förs alltid när man börjar med ett nytt system eller språk. För att kunna göra programmet kommer vi att lära oss vårt första kommando 

print()

När vi skriver Kommandot print() kommer datorn att skriva till output det som vi har specifierat mellan parenteserna. Nu om vi vill att datorn ska hälsa ”hello, world!” till oss kan vi skriva följande kommando: 

[1]:

print("Hello, world!")

Hello, world!

Vi märker nu att programmet har orsakat att programmet har skrivit ”hello,wold!” till ouptutten, just så som vi ville. Men varför fungerar månne inte följande? 

[2]:

print(hello,wold!)

  File "<ipython-input-2-309a55b00953>", line 1
    print(hello,wold!)
                    ^
SyntaxError: invalid syntax

Vad hände – Programmet har skrivit nåt men inte det som vi ville. Utan nu har vi bekantant oss med vår första error. Errors kommer i många olika smaker och vi kommer garanterat att stöta på flera när vi kommer vidare. Vad gör man nar man stöter på en error? Errormeddelande berättar till användaren att något har gåt fel med programmet. Meddelandet berättar till oss oftast vad som har gåt fel och vi som programmerare kan sedan konsultera med vår anka och sedan gå och göra ändringar till vår kod så att den fungerar. Errorn som vi nu mötte var en såkallad SyntaxError vilket berättar till oss att Python kan inte upfatta det som vi har get det.

Men var gick allt fel? Ända skillnade mellan programmet som fungerade och programmet som inte fungerade var karaktaren ” . Det här karaktäret har en speciel betydelse. Allt som vi sätter mellan två ” ” så kommer Python att tolka som text. Om vi inte har dessa symboler med så kommer Python att tolka det som vi sker i parenteserna som mera instruktioner och eftersom vi inte har definiera varken hello eller wolrd som instruktioner så vet inte Python vad de ska göra och berätta till oss detta genom ett error meddelande.

Variabler ¶

Variabler är kanske bekanta för oss får matematik lektionen men nu kommer vi att gå djupare in på vad en variabel egentligen är. Namnet ”variabel” kanske ger oss vår första ledtråd, det är nåt som man kan ändra på. Men för vårt ändamål så måst vi definiera ett variabel lite bredare. Vi kan tänka oss att ett varibel är som en låda. Vi kan kolla in i lådan och titta vad det finns inne i den, vi kan ta ut det som finns i den, vi kan lägga till saker i lådan, vi kan ta bort saker från lådan, samt vi kan byta ut saker som finns i den. Vi kan också namnge lådan och ge det ett ”variabel namn”, det är som om nån skulle ha skrivit på lådan för hjälpa den som kommer sökandes om dess innehåll. Variabeln är kanske en av det mest grundläggande byggkloss som det finns i ett programmeringspråk. I Python så kan en variabel definieras på följande sätt: 

text = "Hejsan,världen!"

Vad beskriver detta? I mitten av kommandot har vi ett likamed(”=”) tecken och med hjälp av den så har vi definierat ett varibel dvs. vi har skapat en låda dot vi kan sätta saker. På vänstra sidan av likamed tecknet har vi vårt variabelnamn i vårt fall text . Det här kan tolkas som en lapp på lådan som berättar oss vad lådan innehåller. På högra sidan av likamed tecknet så har vi det som vi vill sätta i lådan. Vi ser att vi har använt ” ” tecken vilket betyder att vi sätter i lådan nånting som datorn upfattar som text, i detta fall hälsningen ”Hejsan,världen!”
