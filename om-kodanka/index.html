Om Kodanka ¶

Kodanka är ett projekt för att lära ut programmering på svenska. Att koda kan kännas främmande, men det är i slutändan lite som ett logiskt spel som man har nytta av. Vårt mål är att få det att kännas roligt att utforska de möjligheter programmeringen ger, och kom ihåg att ankor kan faktiskt hjälpa.

Själva sidan är gjord i samma språk som lärs ut d.v.s Python. Sphinx och Read the Docs allmänt för att dokumentera kod.
